Evolução da Arquitetura e Tech Stack
Frontend: React com TypeScript e Tailwind CSS.

Backend: Node.js (integração via Fetch API).

Banco de Dados: Migração estratégica de MongoDB para PostgreSQL para garantir maior integridade dos dados e facilitar a modelagem de relações complexas de agendamento.

Refatoração: O código evoluiu de um arquivo único (App.tsx) para uma Arquitetura de Componentes Modulares, facilitando a manutenção e o reaproveitamento de código.
